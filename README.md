## Description

MongoDB is a popular choice for many social networks due to its speed with large amounts of data and flexibility with unstructured data. Social network application is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This uses Express.js for routing, a MongoDB database, and the Mongoose ODM.

Clone the repository and run command `npm install`.
